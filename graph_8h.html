◆ duplicateGraph()

Graph * duplicateGraph ( const Graph *  G)

Perform a deep copy of a graph.

Create a new graph whose connections are exactly the same as G, but whose adjacency list is full of newly allocated nodes. This way the new graph has complete control of the elements that it points to

Parameters
GThe Graph to duplicate
Returns
A pointer to the newly allocated copy
